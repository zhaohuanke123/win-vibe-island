组件层级树

AnimatedOverlay ← 驱动窗口大小同步 (framer-motion)
├─ Overlay ← 编排者：拖拽、吸附、模式切换
│ ├─ NotchRow ← 紧凑条 (始终可见)
│ │ ├─ BarsGlyph ← 24×24 SVG 活动指示
│ │ ├─ AgentDot ← agent 图标 14×14
│ │ └─ Label + RightSlot ← 文本 + 扩展区
│ ├─ [Expanded Panel]
│ │ ├─ PanelHead ← 面板标题栏
│ │ ├─ SessionList
│ │ │ └─ SessionRow × N ← 单会话行 (含 Chevron)
│ │ │ └─ SessionDetail ← 展开的会话详情
│ │ ├─ ApprovalPanel ← 审批请求 (approval focus 时突出)
│ │ └─ SettingsPanel ← 设置 (内嵌)
│ ├─ HookStatus ← Hook 连接指示
│ └─ JumpToast ← 跳转提示浮层
├─ Pill ← 独立模式容器 (notch/panel/notif 三种 mode)
└─ ControlCenter ← 独立窗口 (设置+调试入口)

事件 & 数据流

IN session_start → session_end → Overlay, NotchRow, SessionList
IN state_change → StatusDot, BarsGlyph, StateIndicator
IN tool_use → permission_request → SessionRow (tool list), ApprovalPanel
OUT respond_approval ApprovalPanel → Hook Server → Agent
OUT update_overlay_size AnimatedOverlay → Win32 Window (16ms throttle)
IN hook_status_change → HookStatus
