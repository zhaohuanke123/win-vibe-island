# Notification Sounds

This directory contains notification sound files for Vibe Island.

## Required Files

Place the following WAV files in this directory:

- `pop.wav` - Pop sound
- `ping.wav` - Ping sound
- `glass.wav` - Glass sound
- `hero.wav` - Hero sound (default)
- `blow.wav` - Blow sound
- `bottle.wav` - Bottle sound
- `frog.wav` - Frog sound
- `funk.wav` - Funk sound
- `morse.wav` - Morse sound
- `purr.wav` - Purr sound
- `tink.wav` - Tink sound

## Sound Sources

You can download free notification sounds from:
- [Freesound.org](https://freesound.org/)
- [Mixkit](https://mixkit.co/free-sound-effects/notification/)

## Format

- Format: WAV
- Sample rate: 44100 Hz (recommended)
- Channels: Mono or Stereo
- Duration: 0.5-2 seconds (short notification sounds)

## Build Integration

These sound files will be copied to the output directory during build.
See `tauri.conf.json` for bundle resources configuration.
